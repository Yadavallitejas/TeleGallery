# Security Policy

## Supported Versions

Currently, only the latest release of TeleGallery is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| > 1.0.0 | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please do **NOT** report security vulnerabilities via public GitHub issues.

Instead, please send an email to security@telegallery.app with a description of the issue. We will respond within 48 hours to coordinate a fix and release schedule before public disclosure.

We take the security of your Telegram sessions and local data extremely seriously.
